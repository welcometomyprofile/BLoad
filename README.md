**BLOAD** is a modern GUI for `yt-dlp` built with Python and `customtkinter` for high-quality media downloading.

##  Features
* **Multi-language:** Supports English, Russian, and Ukrainian.
* **Quality Options:** Select resolutions from 360p up to 4K.
* **Smart UI:** Features a "Paste" button and supports `Ctrl+V` / `Ctrl+A` hotkeys.
* **Progress Tracking:** Real-time download speed and percentage display.
* **Bypass Tools:** Uses custom User-Agents and `cookies.txt` support to avoid blocks.
